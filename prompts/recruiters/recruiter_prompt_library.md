# Recruiter Prompt Library

## 1. Recruiter Profile Optimization Prompt

Use this prompt to improve a recruiter’s LinkedIn profile for visibility and credibility.

Prompt:
"Act as a LinkedIn profile strategist for a recruiter. Rewrite my headline, About section, and experience bullets to highlight sourcing expertise, stakeholder management, relationship building, and recruiting impact. Include keywords relevant to talent acquisition, employer branding, and hiring strategy."

## 2. Talent Brand Positioning Prompt

Prompt:
"Create a LinkedIn profile narrative for a recruiter who wants to be seen as a talent advisor and employer brand partner. Emphasize trust, market insight, candidate experience, and measurable recruiting outcomes."
