# LinkedIn Optimizer

LinkedIn Optimizer is a starter repository for creating a highly optimized LinkedIn profile using prompts, templates, and examples. It is designed for job seekers, recruiters, founders, and professionals who want a stronger online presence.

## What this project includes
- A clean landing page for the project
- Prompt libraries for job seekers, recruiters, and founders
- Example before-and-after profile rewrites
- A reusable profile brief template
- A final checklist for profile optimization

## Project structure
- index.html — landing page
- styles.css — page styling
- prompts/ — prompt packs by audience
- templates/ — reusable profile writing templates
- examples/ — before/after rewrite examples
- checklist/ — final review checklist

## Quick start
1. Open index.html in your browser to view the landing page.
2. Browse the prompt files in prompts/.
3. Copy a prompt into ChatGPT, Copilot, Claude, or another AI writing assistant.
4. Fill in your profile details and refine the results.

## Example use cases
- Improve a headline for recruiter visibility
- Rewrite an About section to sound more persuasive
- Turn generic experience bullets into achievement-based statements
- Build a clearer profile for a career transition

## Deployment on GitHub Pages
This project can be published directly via GitHub Pages.

### Steps
1. Push this repository to GitHub.
2. Open your repository on GitHub.
3. Go to Settings → Pages.
4. Under Source, choose Deploy from a branch.
5. Select the main branch and the root folder.
6. Save and wait for the site to publish.

Your site will be available at:
https://<your-username>.github.io/<your-repository-name>/
